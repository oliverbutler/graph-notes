# Xeo Editor 🔐 📝

- 📝 Rich Text Editor
- 🔒 Self-Hostable and Private
- 📱 Mobile-Friendly

## Features

- Rich Text Blocks (Headings, Quotes, Code, Callout, Mentions)
- Mentions between pages
- Graph View
- Favourites
- Page Emojis
- Backlinks
- Dark Mode! 🦉
